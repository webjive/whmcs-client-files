# Client Files for WHMCS

A secure file sharing addon that provides each client with their own private file manager.

## Quick Start

1. Extract to your WHMCS root directory
2. Download elFinder 2.1.64 and extract to `modules/addons/clientfiles/elfinder/`
3. Activate the addon in WHMCS Admin > Configuration > Addon Modules
4. Create file areas for clients from their Summary page

## elFinder Download

This addon requires elFinder. Download from:
https://github.com/Studio-42/elFinder/releases/tag/2.1.64

Extract to: `modules/addons/clientfiles/elfinder/`
